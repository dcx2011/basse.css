What is basse?

Basse is a collection of CSS classes that can be used as a starting point for any web project. It's focused on the non-layout elements, and contains some tips to avoid the "mass-reproduction of classes".

The idea behind

Everybody knows that when you use a <h1> element you are going to have a bigger heading title than when you use an <h3>: it has a semantical meaning that is is generally translated to the design. basse tries to use that same logic with other elements like paragraphs or buttons. It is easy for a developer to remember that the biggest paragraph is <p class="p1"></p>. For example, the paragraph that you are reading is a p3, just bigger than normal, smaller than a p2.

As a convention, in basse the 'normal' size is 4. Everything below 4 (3,2,1) is 'big', and 5 or 6 are used for 'smaller' sizes. 2 and 1 sizes are intended to be highly customizable, but for the rest is recommended to play only with sizes and minor changes.

On the other hand, not all sizes are required. It will depend on the design, but we recommend to have at least 3 sizes: 1, 4 and 6. All the different elements (headings, paragraphs, buttons) should match with each other when used on the same size: try to respect proportions.
